# MindTrace

MindTrace is an AI-driven emotional pattern analysis system designed to help users understand their mood trends and process past experiences.

---

## 🚀 Current Features (Day 2)

* 🔐 User Authentication (Email & Password)
* 📊 Mood Tracking (stored per user)
* ☁️ Firebase Firestore Integration
* 🔄 Real-time Mood Fetching on Home Screen

---

## 🧱 Tech Stack

* **Frontend:** Flutter
* **Backend:** Firebase (Firestore + Auth)
* **Language:** Dart
